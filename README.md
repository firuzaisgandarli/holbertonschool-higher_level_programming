# Python - Import & Modules

Bu proje, Python'da **fonksiyonları import etme**, **modüller oluşturma** ve **komut satırı argümanları** kullanma konularını öğretir.

## Genel Konular
- Python neden harika
- Başka dosyadan fonksiyon import etme
- Import edilen fonksiyonları kullanma
- Modül oluşturma
- `dir()` fonksiyonu kullanımı
- `if __name__ == "__main__":` ile import edildiğinde kodun çalışmasını engelleme
- Komut satırı argümanları ile program yazma

## Gereksinimler
- Editörler: vi, vim, emacs, nano
- Tüm dosyalar Ubuntu 22.04 LTS üzerinde python3 (3.10.*) ile çalıştırılacak
- Dosyaların ilk satırı: `#!/usr/bin/python3`
- Dosyalar newline ile bitmeli
- Kodlar **pycodestyle (2.7.\*)** kurallarına uygun olmalı
- Tüm dosyalar çalıştırılabilir olmalı (`chmod +x`)
- Dosya uzunluğu `wc` ile test edilecek
