# Python - Input/Output

This project is part of the **Holberton School Higher Level Programming** curriculum.  
It focuses on learning how to handle files, input/output operations, and serialization in Python.

## Requirements
- Python 3.8.5 (Ubuntu 20.04 LTS)
- All files must end with a new line
- The first line of all files should be exactly `#!/usr/bin/python3`
- Code must follow **pycodestyle** (version 2.7.*)
- All files must be executable
- Test files must be inside the `tests` folder
- Test files should be text files (`.txt`) and executed with:
  ```bash
  python3 -m doctest ./tests/*
