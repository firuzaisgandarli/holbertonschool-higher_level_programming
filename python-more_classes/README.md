## Project: Python - More Classes

This repository contains tasks for learning **Object-Oriented Programming (OOP)** in Python, focusing on classes and objects.

### Requirements
- All files are interpreted/compiled on Ubuntu 20.04 LTS using Python3 (version 3.8.5).
- First line of all files: `#!/usr/bin/python3`
- Code follows **PEP8** style guide (pycodestyle 2.7.*).
- All files are executable.
- Each file ends with a new line.

### Tasks
- **0. Simple Rectangle**: Write an empty class `Rectangle` that defines a rectangle.
- More tasks build on this, adding attributes, methods, and functionality.
